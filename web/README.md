```
django
├─ apps (подключаемые приложения)
├─ assets (результат сборки фронтенда)
├─ logs (логи по категориям важности)
├─ media (сохраненные картинки)
├─ settings (файлы настроек)
├─ static (собранные статические файлы)
├─ manage.py (точка входа)
├─ requirements.txt (список внешних зависимостей)
├─ urls.py (маршрутизация)
└─ wsgi.py (точка входа для wsgi-сервера)
```
